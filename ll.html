IMAGE CONVERTER GUI - SINGLE FILE

โปรแกรมแปลงรูป + ปรับขนาดรูป
ออกแบบมาให้ใช้ง่าย คลิกไม่กี่ครั้ง เหมือนโปรแกรมยุคเก่า

ทำงานทั้งหมดในเครื่องคุณ (Local only)
❌ ไม่อัปโหลดรูปไปที่ไหน
❌ ไม่มีโฆษณา / tracking / แอบส่งข้อมูล

✅ สิ่งที่โปรแกรมทำได้

1️⃣ เลือกรูป (ทีละ 1 ไฟล์)

  • เลือกรูปจากเครื่อง (PNG / JPG / WEBP / BMP / TIFF / ICO)
  • หรือโหลดจาก URL
  • คลิกช่อง URL → คลิกขวา → วางลิงก์
  • แสดงชื่อไฟล์แบบสั้น ไม่รก

2️⃣ แปลงนามสกุลรูป (Format)

  • PNG
  • JPG / JPEG
  • WEBP
  • BMP
  • TIFF
  • ICO (128×128) สำหรับทำไอคอน
  • PDF (รูปเดียว)

3️⃣ ปรับคุณภาพรูป (Quality)

  • ใช้กับ JPG / WEBP
  • ลดขนาดไฟล์ได้มาก

4️⃣ Resize รูป (Preset เดียว)

  • ใส่ Width / Height (เว้นว่าง = ไม่ resize)
  • Fit – คงสัดส่วน (แนะนำ)
  • Stretch – ยืดตามขนาด
  • Fill – เต็มกรอบ + ครอปกลาง
  • จำกัดสูงสุด 3840×2160

5️⃣ บันทึกผลลัพธ์

  • 📁 บันทึกลงโฟลเดอร์
  • 📦 บันทึกเป็น ZIP
  • ชื่อไฟล์อัตโนมัติ: ชื่อเดิม_ขนาด_เวลา.ext

6️⃣ ทำเป็นไฟล์ .exe

  • เครื่องอื่นไม่ต้องลง Python
  • ดับเบิลคลิกใช้งานได้ทันที

❌ สิ่งที่ตั้งใจไม่ใส่

  • เลือกหลายไฟล์พร้อมกัน
  • Preset หลายชุด
  • อัปโหลดขึ้นเซิร์ฟเวอร์
  • โฆษณา / แอด / tracking

🧠 เหมาะกับใคร

  • คนอยากแปลงรูปเร็วๆ
  • ทำไอคอน .ico
  • ลดขนาดรูปก่อนอัปโหลด
  • แจกโปรแกรมเป็นไฟล์ .exe
  • ไม่อยากใช้เว็บแปลงรูป
